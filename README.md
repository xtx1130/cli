# leju-cli 

> cli for leju


## What a fuck,I wanna to establish a folder quickly and use plugins as my wish

### What is leju?

Establish folders quickly
Use babels quickly

```bash
npm install -g leju-cli
```

### parameter

```bash
leju -h --help 
```

```bash
leju version
```

```bash
leju folder
```

```bash
leju update
```

```bash
leju babel (-m)
```
